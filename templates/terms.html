4Accuracy & Disclaimer

URLiar uses a machine learning model trained on publicly available URL datasets. While we strive for accuracy, the system may produce:

  • False positives — legitimate websites flagged as phishing
  • False negatives — phishing websites classified as safe
🔍 Always verify suspicious URLs using multiple security tools (e.g., Google Safe Browsing, VirusTotal) before making any decisions based on URLiar's output.
